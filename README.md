# WeatherClearFix

Are you tired of rain 5 minutes after using the /weather clear command? Well this is a plugin to fix exactly that!
## Features:
- Extremely lightweight
- Vanilla like
- Configurable
- Open source
## Config
```
main-world: world
weather-duration: #In ticks
   max: 100000
   min: 50000
```
## Usage:
- /weather clear
